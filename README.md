# paraglider

First Upload:

Das Herzstück des Spiels ist die Klasse

>>> ParagliderMainScript

dieses liegt genau ein mal auf dem GameObjekt

>>> Paraglider Game Controller (DontDestroyOnLoad) @ Scene paragliderStandby


in dem script ist oben die liste mit Methoden und die Variablen mit Kommentaren
zu Beginn einfach mal starten und lesen, was da in Pink auf dem Bildschirm steht

dann reset auslösen [X]
dann erstes Level laden [L]
dann fliegen (wooosh)
wenn (crash) dann Respawn [R]
